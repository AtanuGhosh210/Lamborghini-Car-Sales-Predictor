Predicting Lamborghini Car Sales using Regression Analysis

This project uses supervised learning techniques to predict future sales growth of Lamborghini cars based on historical sales data.
The goal is to build a linear regression model to accurately estimate the growth percentage given past sales data.
Dataset Source: Kaggle (Lamborghini Sales and Stocks)

from sklearn.model_selection import train_test_split

# Splitting the data into training and testing sets
X = data[['Sales']].values  # Independent variable (Sales)
y = data['Growth'].values   # Dependent variable (Growth)

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=0)
In[6]:
from sklearn.linear_model import LinearRegression

# Creating the linear regression model
model = LinearRegression()

# Training the model
model.fit(X_train, y_train)

# Predicting on the test data
y_pred = model.predict(X_test)
